
Instructions to Use This Portfolio Template:

1. Open 'index.html' in your browser to preview your site.
2. Edit index.html to update your name, bio, skills, and projects.
3. Edit style.css to change design (colors, fonts, layout).
4. Upload all files to a GitHub repository.
5. Go to GitHub > Settings > Pages > Set source to 'main' and folder to '/' to deploy it.

Your website will be live at: https://yourusername.github.io/your-repo-name/
